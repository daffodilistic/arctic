An APK that uses the Android Security Squad's Zip Extra wraparound bug and the engineer mode to root the Kobo Arc tablet without connecting it to a computer.

#Build

```
cd doublerainbow
ant clean debug
cd ..
./builddoublerainbow.sh
cd arctic
ant clean release
```
